# 滑动窗口

滑动窗口题关注的是连续区间上的状态维护。重点不是“窗口本身”，而是窗口内部要维护什么不变量，以及在什么条件下扩张或收缩。

## 无重复字符的最长子串

!!! problem "[3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/?envType=study-plan-v2&envId=top-100-liked){target=_blank}"

    给定一个字符串 `s`，请找出其中不含有重复字符的最长子串，并返回该子串的长度。

    子串必须是连续的字符区间。题目要求的是长度的最大值，而不是输出具体子串本身。

窗口内的不变量是“字符互不重复”。设窗口为 `[left, right]`，当右端点加入一个新字符时，如果这个字符在当前窗口中已经出现过，就必须把左端点移动到它上一次出现位置的后面。

换句话说，窗口只需要记住每个字符最近一次出现的位置。若字符 `c` 的上次位置是 `pos`，并且 `pos >= left`，那么新的合法左边界就是：

$$
left = pos + 1
$$

这样窗口始终保持无重复字符，且每个字符只会进入和离开窗口有限次，因此总复杂度是线性的。

## 找到字符串中所有字母异位词

!!! problem "[438. 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/description/?envType=study-plan-v2&envId=top-100-liked){target=_blank}"

    给定两个字符串 `s` 和 `p`，请找出 `s` 中所有与 `p` 互为字母异位词的子串，并返回这些子串的起始下标。

    这里要求返回的是所有满足条件的起始位置，结果顺序按扫描顺序输出即可。只要某个长度与 `p` 相同的连续子串与 `p` 的字符组成完全一致，就应当记录它的起始下标。

这题的窗口长度是固定的，始终等于 `|p|`。因此不需要处理“窗口是否过长”的动态问题，只需要维护当前窗口和目标字符串之间的字符计数差。

一种常见写法是维护 `need`：

- 初始化时统计 `p` 中每个字符还需要多少次。
- 右端点加入字符时，把对应计数减一。
- 左端点移出字符时，把对应计数加一。

若某个字符加入前计数仍然大于 `0`，说明它填补了一个缺口；若移出后计数重新大于 `0`，说明窗口又缺少了这个字符。

当窗口长度等于 `|p|` 且所有缺口都被填平时，当前窗口就是一个异位词。
